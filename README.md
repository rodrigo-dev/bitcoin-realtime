## Quick Start

#### Start Developing

```sh
$ git clone https://github.com/rodrigo-dev/bitcoin-realtime
$ cd bitcoin-realtime
$ npm install
$ npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
